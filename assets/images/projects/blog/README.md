# Imágenes del Proyecto: Página Web de Blogs

Esta carpeta contiene las imágenes de la galería para el proyecto "Página Web de Blogs".

## 📁 Cómo agregar imágenes

1. **Agrega las imágenes** con nombres numerados secuencialmente:
   - `1.jpg` (primera imagen)
   - `2.jpg` (segunda imagen)
   - `3.jpg` (tercera imagen)
   - Y así sucesivamente...

2. **Formatos soportados:**
   - JPG (recomendado)
   - PNG
   - WebP

## 📐 Tamaño recomendado para las imágenes

- **Tamaño recomendado:** 1200x800 píxeles (relación 3:2)
- **Tamaño mínimo:** 800x600 píxeles
- **Peso máximo:** 500 KB por imagen (comprimidas)
- **Relación de aspecto:** 3:2 o 16:9 funcionan mejor

## 🎨 Imagen principal del proyecto

La imagen principal que se muestra en la tarjeta del proyecto debe estar en:
- `assets/images/projects/blog.jpg`
- Tamaño recomendado: 800x500 píxeles

## ✅ Después de agregar las imágenes

Una vez que agregues las imágenes con los nombres `1.jpg`, `2.jpg`, etc., aparecerán automáticamente en la galería del proyecto cuando hagas clic en "Demo".

## 💡 Tips

- Usa nombres secuenciales sin saltar números
- Mantén el mismo formato para todas las imágenes (por ejemplo, todas JPG)
- Optimiza las imágenes antes de subirlas para mejorar el rendimiento
- Las imágenes deben ser capturas de pantalla o fotos representativas del proyecto
