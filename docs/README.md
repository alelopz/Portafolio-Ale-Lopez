## 🚀 Cómo Usar

### **1. Clonar el Repositorio**
```bash
git clone https://github.com/alelopz/portafolio.git
cd portafolio
```

### **2. Abrir en el Navegador**
Simplemente abre `index.html` en tu navegador web preferido.

### **3. Personalizar (Opcional)**
- Modifica `assets/js/config.js` para actualizar información personal
- Cambia colores en `assets/css/styles.css`
- Agrega nuevas secciones en `index.html`

## 🎨 Personalización

### **Cambiar Información Personal**
Edita `assets/js/config.js`:
```javascript
personal: {
    name: "Tu Nombre",
    title: "Tu Título",
    email: "tu.email@ejemplo.com",
    // ... más configuraciones
}
```

### **Cambiar Colores del Tema**
Modifica las variables CSS en `assets/css/styles.css`:
```css
:root {
    --primary-color: #2563eb;
    --secondary-color: #7c3aed;
    --accent-color: #f59e0b;
    /* ... más colores */
}
```

### **Agregar Nuevos Proyectos**
En `assets/js/config.js`:
```javascript
projects: [
    // ... proyectos existentes
    {
        id: "nuevo-proyecto",
        title: "Nuevo Proyecto",
        description: "Descripción del proyecto",
        technologies: ["HTML", "CSS", "JavaScript"],
        github: "https://github.com/usuario/proyecto",
        demo: "https://demo.com"
    }
]
```

## 🌐 Despliegue

### **GitHub Pages**
1. Sube tu código a GitHub
2. Ve a Settings > Pages
3. Selecciona la rama principal
4. Tu portafolio estará disponible en `https://usuario.github.io/portafolio`

### **Hosting Tradicional**
1. Sube todos los archivos a tu servidor web
2. Asegúrate de que `index.html` esté en la raíz
3. Verifica que las rutas de archivos sean correctas

## 🔧 Mantenimiento

### **Actualizar Proyectos**
1. Modifica `assets/js/config.js`
2. Agrega nuevas imágenes en `assets/images/projects/`
3. Actualiza `assets/js/image-paths.js` si es necesario

### **Actualizar Habilidades**
Edita la sección de skills en `assets/js/config.js`:
```javascript
skills: {
    languages: {
        title: "Lenguajes de Programación",
        items: ["C#", ".NET", "SQL", "JavaScript"]
    }
    // ... más categorías
}
```
